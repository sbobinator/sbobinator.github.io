Core Technologies

NeMo Parakeet

NVIDIA's TDT 0.6B v3 model delivers Italian-first speech recognition with segment timestamps. Loaded once per worker process, it runs on CPU for accessibility or CUDA when you need faster turnaround on long archives.

FastAPI + HTMX

A lightweight operator UI without a heavy SPA. Upload forms, job lists, and per-job progress pages poll the server for live updates. Jinja2 templates keep the stack simple to deploy and audit.

Docker Compose

Package UI and worker for headless Linux mini PCs. The ASR model can be baked into the image; local Qwen downloads at startup when RAM allows. Default port mapping 8502:8501 for home-lab use.

WORKER_DAEMON // V1.0

The Background Worker.

NeMo cannot safely load inside the web server's request thread on Windows — so transcription runs in a dedicated worker process. It claims the next queued job, executes the full pipeline, and loops until stopped.

On startup the worker recovers orphaned running jobs and reconciles disk folders with SQLite — so a crash mid-transcription does not leave your history in an inconsistent state.

  • Orphan recovery on boot
  • Disk ↔ database reconcile
  • Chunked ASR for long files
worker.py 
while not _stop_event.is_set():
    job = claim_next_job()
    if job is None:
        _stop_event.wait(poll_interval)
        continue

    logger.info("Processing job %s", job.id)
  try:
        run_pipeline(job.id)
  except Exception:
        logger.exception("Job %s failed", job.id)
