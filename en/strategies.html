Module_01

Parakeet ASR

The transcription engine. NVIDIA NeMo Parakeet TDT 0.6B v3 runs entirely on your hardware — Italian-first, EU multilingual capable, with timestamped segments for subtitle export. Long files are split into chunks, transcribed sequentially, and deduplicated at boundaries.

Model parakeet-tdt-0.6b-v3
Runtime CPU or CUDA
Input MP4, MKV, WAV, MP3…
Pre-process ffmpeg → mono WAV
src/sbobinator/transcribe.py
def transcribe(input_path, config, work_dir, on_progress): model = _get_model(config) wav_path, _ = prepare_audio(input_path, work_dir=work_dir) duration = get_duration_sec(wav_path) if duration <= config.chunk_threshold_sec: return _transcribe_file(model, wav_path) chunks = split_audio_chunks(wav_path, chunk_dir, config.chunk_length_sec) for i, chunk in enumerate(chunks): on_progress("transcribe", pct, f"chunk {i+1}/{len(chunks)}") partial = _transcribe_file(model, chunk) segments.extend(partial.segments) return TranscriptResult(segments=segments, text=_merge_text(segments))

Module_02

Job Queue

Not a one-shot upload dialog. Sbobinator tracks every file in SQLite, shows live progress per job, keeps history at /jobs, and lets you reprocess or delete entries. On startup the worker recovers orphaned runs and reconciles missing folders with the database.

Storage SQLite + data/output/
Worker subprocess / Docker
Detail page /jobs/{id}
Batch multi-file upload
Module_03

LLM Summary

When you need more than raw transcript text. Configure a provider in the web UI — DeepSeek, OpenAI, Claude, Gemini, Kimi for speed, or download Qwen for a fully offline summary step. Executive and detailed modes adapt the prompt to meetings, interviews, or long-form content.

Settings /settings/summary
Cloud your API keys
Local Qwen RAM ≥ 16 GB
Default disabled
CLI with summary
# Transcription only — no cloud sbobina transcribe meeting.mp4 -o data/output # Optional cloud summary (your key) sbobina transcribe podcast.wav -s \ --summary-provider deepseek # Web UI: configure once, queue many jobs sbobina ui # → http://localhost:8501/jobs
