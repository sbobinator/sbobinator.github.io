01. Pipeline

Processing Flow

01

Upload

Drop MP4, MKV, WAV, MP3 and more via the FastAPI web UI or CLI. Jobs are tracked in SQLite with live progress.

FastAPI / HTMX / SQLite
02

Extract

ffmpeg normalizes audio from video containers. Long files are chunked for stable GPU/CPU transcription.

ffmpeg / pydub
03

Transcribe

NVIDIA NeMo Parakeet TDT 0.6B v3 runs locally for Italian speech recognition with timestamped segments.

NeMo / CUDA or CPU
04

Export

Download TXT, SRT, and optional LLM summaries. Reprocess or delete jobs from the history UI.

DeepSeek / OpenAI / Qwen

02. Modules

Core Features

Three modules — durable job queue, local Parakeet ASR, optional LLM summaries — plus Docker for always-on mini-PC deployment.

MOD_01

Job Queue

Upload multiple files, track status live, open each job detail page, reconcile disk state, reprocess or delete entries. Built for long recordings and batch workflows.

  • /jobs list + filters
  • /jobs/{id} live progress
  • Disk ↔ DB reconcile
Typical Outputs
transcript.txt plain text
subtitles.srt timed cues
summary.md optional
Docker port 8502 → UI
Summary Providers
Cloud APIs DeepSeek, OpenAI…
Local Qwen offline
RAM (local LLM) ≥ 16 GB
ASR model size ~2.5 GB
MOD_02

LLM Summary

Summaries are strictly opt-in. Configure providers in the web UI, use cloud APIs for speed, or download Qwen for fully offline pipelines on capable hardware.

  • Provider registry
  • Detailed vs executive modes
  • Settings page /settings/summary
MOD_03

Parakeet Engine

The ASR core: NVIDIA NeMo Parakeet TDT 0.6B v3, ~2.5 GB download, Italian-first with EU multilingual support. ffmpeg extracts audio; long files are chunked and merged with boundary deduplication.

  • CPU or CUDA inference
  • Timestamped SRT export
  • CLI + worker pipeline

03. Privacy

Local First

Transcription stays on your machine. Cloud APIs are only invoked when you explicitly enable LLM summaries with your own keys.

No Mandatory Cloud

ASR does not phone home. Models are downloaded once and cached under your data directory.

Mini-PC Ready

Docker Compose packages UI + worker for headless home-lab deployment on Linux.

License Gate

First-run acknowledgment for personal use. Commercial terms documented in LICENSE and MkDocs.
