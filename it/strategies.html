Modulo_01

Parakeet ASR

Il motore di trascrizione. NVIDIA NeMo Parakeet TDT 0.6B v3 gira interamente sul tuo hardware — italiano-first, multilingue EU, con segmenti timestampati per export sottotitoli. I file lunghi vengono spezzati in chunk, trascritti in sequenza e deduplicati ai confini.

Modello parakeet-tdt-0.6b-v3
Runtime CPU o CUDA
Input MP4, MKV, WAV, MP3…
Pre-process ffmpeg → WAV mono
src/sbobinator/transcribe.py
def transcribe(input_path, config, work_dir, on_progress): model = _get_model(config) wav_path, _ = prepare_audio(input_path, work_dir=work_dir) duration = get_duration_sec(wav_path) if duration <= config.chunk_threshold_sec: return _transcribe_file(model, wav_path) chunks = split_audio_chunks(wav_path, chunk_dir, config.chunk_length_sec) for i, chunk in enumerate(chunks): on_progress("transcribe", pct, f"chunk {i+1}/{len(chunks)}") partial = _transcribe_file(model, chunk) segments.extend(partial.segments) return TranscriptResult(segments=segments, text=_merge_text(segments))

Modulo_02

Coda Job

Non una finestra upload one-shot. Sbobinator traccia ogni file in SQLite, mostra progress live per job, mantiene cronologia su /jobs, e permette riprocessare o eliminare. All'avvio il worker recupera run orfane e riconcilia cartelle mancanti col database.

Storage SQLite + data/output/
Worker subprocess / Docker
Pagina dettaglio /jobs/{id}
Batch upload multi-file
Modulo_03

Riassunto LLM

Quando serve più del testo grezzo. Configura un provider nella web UI — DeepSeek, OpenAI, Claude, Gemini, Kimi per velocità, oppure scarica Qwen per un passo riassunto completamente offline. Modalità executive e detailed adattano il prompt a riunioni, interviste o contenuti lunghi.

Impostazioni /settings/summary
Cloud le tue API key
Qwen locale RAM ≥ 16 GB
Default disabilitato
CLI con riassunto
# Solo trascrizione — niente cloud sbobina transcribe riunione.mp4 -o data/output # Riassunto cloud opzionale (tua chiave) sbobina transcribe podcast.wav -s \ --summary-provider deepseek # Web UI: configura una volta, accoda molti job sbobina ui # → http://localhost:8501/jobs
