01. Pipeline

Flusso di Elaborazione

01

Upload

Carica MP4, MKV, WAV, MP3 dalla Web UI FastAPI o da CLI. I job sono tracciati in SQLite con avanzamento live.

FastAPI / HTMX / SQLite
02

Estrazione

ffmpeg normalizza l'audio dai container video. I file lunghi vengono spezzati in chunk per una trascrizione stabile.

ffmpeg / pydub
03

Trascrizione

NVIDIA NeMo Parakeet TDT 0.6B v3 in locale per riconoscimento vocale italiano con segmenti timestampati.

NeMo / CUDA o CPU
04

Export

Scarica TXT, SRT e riassunti LLM opzionali. Rielabora o elimina job dallo storico.

DeepSeek / OpenAI / Qwen

02. Moduli

Funzionalità Principali

Tre moduli — coda job resiliente, ASR Parakeet locale, riassunti LLM opzionali — più Docker per deploy always-on su mini PC.

MOD_01

Coda Job

Carica più file, segui lo stato in tempo reale, apri il dettaglio di ogni job, riconcilia disco e DB, rielabora o elimina. Pensato per registrazioni lunghe e batch.

  • Lista /jobs + filtri
  • /jobs/{id} avanzamento live
  • Riconciliazione disco ↔ DB
Output tipici
transcript.txt testo piano
subtitles.srt cue temporali
summary.md opzionale
Porta Docker 8502 → UI
Provider riassunto
API cloud DeepSeek, OpenAI…
Qwen locale offline
RAM (LLM locale) ≥ 16 GB
Modello ASR ~2,5 GB
MOD_02

Riassunto LLM

Il riassunto è sempre opt-in. Configura i provider nella Web UI, usa le API cloud per velocità, oppure scarica Qwen per pipeline completamente offline.

  • Registry provider
  • Modalità dettagliata / executive
  • Pagina /settings/summary
MOD_03

Motore Parakeet

Il core ASR: NVIDIA NeMo Parakeet TDT 0.6B v3, download ~2,5 GB, italiano-first con supporto multilingue EU. ffmpeg estrae l'audio; i file lunghi vengono spezzati in chunk e uniti con deduplicazione ai confini.

  • Inferenza CPU o CUDA
  • Export SRT con timestamp
  • Pipeline CLI + worker

03. Privacy

Local First

La trascrizione resta sulla tua macchina. Le API cloud vengono chiamate solo se abiliti esplicitamente il riassunto LLM con le tue chiavi.

Nessun cloud obbligatorio

L'ASR non invia dati in rete. I modelli si scaricano una volta e restano in cache locale.

Pronto per mini PC

Docker Compose impacchetta UI + worker per deploy headless su Linux.

Gate licenza

Accettazione al primo avvio per uso personale. Termini commerciali in LICENSE e MkDocs.
