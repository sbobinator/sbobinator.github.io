01. Come Funziona

Quattro Passi Semplici

Nessun abbonamento. Installi una volta, apri l'app nel browser e trasformi le registrazioni in file che puoi cercare, citare e sottotitolare.

01

Carica

Trascina una riunione, un episodio podcast, un'intervista o qualsiasi file MP4, MKV, WAV, MP3. Puoi caricarne diversi insieme.

Dal browser, drag & drop
02

Attendi (o vai via)

Sbobinator lavora in background. I file lunghi sono gestiti automaticamente. Chiudi il browser — il lavoro continua.

Progresso sempre visibile
03

Trascrivi

Un motore vocale italiano gira sul tuo PC — non su server di terzi. Ottimizzato per l'italiano parlato reale.

Offline dopo il setup
04

Scarica

Ottieni testo da leggere e cercare, più sottotitoli SRT per Premiere, DaVinci o YouTube. Riassunto AI solo se lo vuoi.

TXT · SRT · riassunto opzionale

03. Privacy

I Tuoi File Restano Tuoi

La trascrizione online significa mandare le tue registrazioni vocali su server altrui. Sbobinator gira sul tuo computer — così call con clienti, note mediche, interviste legali e contenuti personali non devono uscire di casa.

Niente upload per trascrivere

Il motore vocale gira in locale. Dopo il download iniziale del modello, puoi trascrivere anche senza internet.

Sul tuo PC

Windows per l'uso quotidiano, oppure un piccolo server di casa se vuoi tenerlo sempre acceso in rete.

Riassunti AI solo se li attivi

Ti serve un riepilogo? Abilitalo e scegli l'AI. Altrimenti ottieni testo e sottotitoli — punto.
